# Primitive Types

The Jule API contains definitions for primitive types in the `types.hpp` header.

## Variables

```cpp
constexpr std::nullptr_t nil;
```
Equivalent of Jule's `nil`

## Type Aliases

```cpp
jule::I8;
```
Equivalent of Jule's `i8` type.
