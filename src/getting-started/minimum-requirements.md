# Minimum Requirements

## Operating Systems

### Linux

Kernel 3.2 or later.

### macOS (aka Darwin)

macOS Big Sur 11 or newer.

### Windows

Windows 10 and higher or Windows Server 2016 and higher.

## Architectures

### AMD64 (aka x86_64)

All AMD64 CPUs.

### ARM64 (aka AArch64)

All ARM64 CPUs.

### i386 (aka x86, intel 386)

All i386 CPUs.
