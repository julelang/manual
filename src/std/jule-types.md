## RealKindOf
```jule
fn RealKindOf(kind: str): str
```
Returns kind&#39;s bit\-specific kind if bit\-specific like int, uint, and uintptr\. Returns kind if not bit\-specific\. Bit\-size is determined by runtime\.

## BitSizeOf
```jule

```
